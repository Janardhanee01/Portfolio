HIREsuME

This project will help user through the process that can be followed to build their resume using the templates provided which can even be customized. Automated resume builder is a web application which helps the student to get their resume in hand just by filling up a simple form where important credentials need to be filled. The resume is downloadable in pdf format. Also the user can login again to access the previous resume that he had made It helps the user to build their resume by just filling a simple form where important redentials need to be filled. The details filled are embedded with template provided which can also be customized. The finished resume is downloadable as a pdf.

Technologies used

Node js to Process the information on the server(Node) and using some HTML to PDF libraries to generate the resume. ReactJS, React-Boostrap to create an interactive front end HTML & CSS to Develop basic template(s) of the resume
